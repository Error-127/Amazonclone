# Amazon Clone

A full-stack e-commerce platform built with React, Node.js, Express, and MongoDB that replicates the core user experience of Amazon.

## 🚀 Live Demo
* **Frontend (Vercel):** [https://amazonclone-two-alpha.vercel.app](https://amazonclone-two-alpha.vercel.app)

---

## ✨ Features
* **User Authentication:** Secure user signup, login, and token-based state management using JWT and bcryptjs hashing.
* **Dynamic Product Listing:** Real-time database product retrieval featuring a light-blue carousel hero banner and search filtering.
* **Detailed Product Views:** Dedicated custom page views handling both relational numbers and standard MongoDB ObjectIds.
* **Persistent Database Cart:** Cloud-synced user shopping basket tracking product additions, quantity counters, and sub-total sums securely.
* **Checkout Gateway:** A dedicated checkout interface route guiding users smoothly from their basket to completion.

---

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Axios, React Router DOM v6
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas, Mongoose ODM
* **Deployment:** Vercel (Client), Render (Server)
